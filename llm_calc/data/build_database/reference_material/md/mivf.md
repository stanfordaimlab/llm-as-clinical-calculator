## FORMULA

Calculated using the "4-2-1" Rule:

-   For 0-10 kg: 4 mL/kg/hr
-   For \>10-20 kg: 40 mL + 2 mL/kg/hr for every kg \>10 kg
-   For \>20 kg: 60 mL + 1 mL/kg/hr for every kg \>20 kg
